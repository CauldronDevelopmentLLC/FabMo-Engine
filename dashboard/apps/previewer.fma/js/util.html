Checks if two points in 3D are considered as "equal". This function is useful to avoid to be too much precise (JavaScript can not do very precise calculations).

Parameters:

  • a must be an object.
    (Point A.)

  • b must be an object.
    (Point B.)

Returns a boolean
(True if the two points are nearly equal.)

exports.samePosition = function(posA, posB) { return (nearlyEqual(posA.x, posB.x) && nearlyEqual(posA.y, posB.y) && nearlyEqual(posA.z, posB.z)); };
