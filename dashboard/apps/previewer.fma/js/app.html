Resizes the viewer according to the running job footer height. The height should be equal to zero if no job is running.

Parameters:

  • footerHeight must be a number.
    (The footer height.)
function resizeAccordingFooter(footerHeight) { var width = window.innerWidth; var height = window.innerHeight - $("#topbar").height() - footerHeight; $('#preview').size(width, height); viewer.resize(width, height); }

Resizes the viewer. Function to call when the window has been resized.

Initializes the handler for updating the live viewer.

Parameters:

  • jobId must be a number.
    (The job id.)
function initializeLiveViewerHandler(jobId) { fabmo.on("status", function(status) { if(status.state !== "running") { return; } if(status.job && status.job._id === jobId && status.line !== null) { viewer.updateLiveViewer(status.line); } }); }

Initializes the viewer object.

Parameters:

  • gcode must be a string.
    (The G-Code to display.)

  • isLive must be a boolean.
    (Sets if the G-Code should be displayed live.)

  • jobId must be a number.
    (The job id.)
